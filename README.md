## プロダクト説明

<!-- プロダクトの説明を入力 -->
待ち合わせや、場所を教えるとき分かりやすい目印となるようなものがあれば、もっと円滑なコミュニケーションが取れるのではないかと考えました。

そこで、私たちが作ったのは、
**ARex（エイレックス）**
です。

**ARex（エイレックス）**
はARを使ったコミュニケーションツールです。Apex Legendsというゲーム内のピンシステムを現実世界（AR上)で使えるようにし、文字に頼らない簡単で円滑なコミュニケーションを取れるように意識しました。
